Fitur

Beranda, Produk, Detail Produk, Keranjang, Checkout  
React Router untuk navigasi  
Responsive design  


Cara Menggunakan

# 1. Install dependencies
npm install

# 2. Jalankan aplikasi
npm run dev

Buka browser: `http://localhost:5173`


## Build Production

npm run build
